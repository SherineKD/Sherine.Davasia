# Sherine K Davasia - Portfolio Website

![Portfolio Screenshot](screenshot.png)

A responsive portfolio website showcasing my skills, projects, and experience as a Web Developer & Graphic Designer.

## Features
- Responsive design (mobile, tablet, desktop)
- Animated sections with Intersection Observer
- Interactive portfolio gallery with filtering
- Lightbox image viewer
- Contact form with validation
- Smooth scrolling navigation
- Dynamic header behavior

## Technologies Used
- HTML5
- CSS3 (with animations and transitions)
- JavaScript (ES6)
- Font Awesome icons
- Google Fonts (Montserrat)

No installation required - this is a static website. Simply clone the repository and open `index.html` in your browser.
